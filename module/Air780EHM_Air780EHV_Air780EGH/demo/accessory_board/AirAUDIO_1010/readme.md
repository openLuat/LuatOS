## 功能模块介绍：

本项目基于AirAUDIO_1010配件板和Air780EHM核心板/Air780EGH核心板，演示语音通话和音频处理功能。包含以下两个主要功能模块，可根据需求选择相应功能进行测试：

**使用中需要注意将AirAUDIO_1010 配件板中PA开关拨到OFF，让软件控制PA，避免pop音**。

1、cc：实现完整的VoLTE语音通话功能，支持多种通话场景和通话管理，详细请看 [cc demo](https://gitee.com/openLuat/LuatOS/tree/master/module/Air780EHM_Air780EHV_Air780EGH/demo/CC/Air780EHM_Air780EGH)

2、audio：提供完整的音频采集、处理和播放功能，支持多种音频格式和操作模式，详细请看 [audio demo](https://gitee.com/openLuat/LuatOS/tree/master/module/Air780EHM_Air780EHV_Air780EGH/demo/audio/Air780EHM_Air780EGH)
