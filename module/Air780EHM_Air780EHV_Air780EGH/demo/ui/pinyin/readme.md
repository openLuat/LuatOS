# pinyin (拼音输入)

1、LuatOS支持pinyin (拼音输入)，通过输入汉字的拼音进行选字。

2、使用pinyin (拼音输入)开发的demo项目可以参考以下个demo：

Air780EHM/Air780EHV/Air780EGH核心板+AirLCD_1010配件板：支持exeasyui各种组件、支持矢量字体、支持触摸控制

- 代码：[module/Air780EHM_Air780EHV_Air780EGH/demo/ui/easyui/single](https://gitee.com/openLuat/LuatOS/tree/master/module/Air780EHM_Air780EHV_Air780EGH/demo/ui/easyui/single)
- 文档：[pinyin (拼音输入)在Air780EHV+AirLCD_1010上的应用](https://docs.openluat.com/air780ehv/luatos/app/multimedia/ui/single/)
- 文档：[pinyin (拼音输入)在Air780EHM+AirLCD_1010上的应用](https://docs.openluat.com/air780epm/luatos/app/multimedia/ui/single/)
- 文档：[pinyin (拼音输在Air780EGH+AirLCD_1010上的应用](https://docs.openluat.com/air780egh/luatos/app/multimedia/ui/single/)

在这个demo中，pinyin (拼音输入)的应用，详见**single**文件夹内的win_input.lua脚本文件
