# UI + UART 独立项目

## 项目简介

这是一个从完整Turnkey开发板项目中独立出来的UI+UART功能演示项目。

## 项目结构

```
uart_ui_project/
├── main.lua                 # 主程序入口
├── app/
│   └── uart_app.lua        # 串口应用模块
├── ui/
│   ├── ui_main.lua         # UI主程序，管理页面切换
│   ├── home_page.lua       # 首页
│   └── uart_page.lua       # 串口通信页面
├── hw_drv/
│   └── lcd_drv.lua         # LCD显示驱动
└── tp_key_drv/
    └── tp_drv.lua          # 触摸面板驱动
```

## 功能特性

### 1. UI系统
- 基于AirUI框架
- 支持页面切换和返回
- 支持触摸交互
- 状态栏显示时间和信号强度

### 2. 串口功能
- 支持UART1/UART2/UART3
- 可选波特率：9600/115200/460800
- 支持文本模式和十六进制模式
- 实时接收数据显示
- 缓冲区最大1000字符

### 3. 页面功能

#### 首页 (home_page)
- 显示当前时间和4G信号强度
- 快速进入串口功能页面

#### 串口页面 (uart_page)
- 串口选择下拉框
- 波特率选择下拉框
- 打开/关闭串口按钮
- 十六进制模式切换
- 发送数据输入框
- 接收数据显示区域（带清空功能）

## 使用说明

### 1. 硬件准备
- Air780EHM开发板
- LCD显示屏（ST7796控制器）
- GT911触摸屏
- 串口设备（可选）

### 2. 编译和烧录
1. 使用Luatools工具
2. 选择项目目录
3. 编译并下载到设备

### 3. 串口使用流程
1. 在首页点击"串口通信"按钮进入串口页面
2. 选择要使用的串口（UART1/UART2/UART3）
3. 选择波特率
4. 点击"打开"按钮打开串口
5. 勾选"十六进制模式"可切换发送/接收格式
6. 在输入框输入数据，点击"发送"按钮发送
7. 接收到的数据会自动显示在下方接收区域
8. 点击"清空"按钮清空接收区
9. 点击"返回"按钮返回首页

## 串口事件接口

### 发布的事件
- `uart_opened(uart_id, baudrate)` - 串口打开成功
- `uart_closed(uart_id)` - 串口关闭
- `uart_sent(uart_id, data)` - 数据发送成功
- `uart_data(uart_id, data)` - 接收到数据

### API接口
```lua
-- 打开串口
uart_app.open(uart_id, baudrate_str)

-- 关闭串口
uart_app.close()

-- 发送文本数据
uart_app.send(data)

-- 发送十六进制数据
uart_app.send_hex(hex_str)

-- 获取波特率选项
uart_app.get_baudrate_options()

-- 获取串口状态
uart_app.get_status()
```

## 注意事项

1. 串口打开后，关闭前不能打开其他串口
2. 十六进制模式下，发送的数据格式为十六进制字符串（如"48656C6C6F"）
3. 接收缓冲区最大保留1000字符，超过自动截断
4. LCD分辨率：480x320，方向：90°
5. 触摸屏使用I2C1接口
