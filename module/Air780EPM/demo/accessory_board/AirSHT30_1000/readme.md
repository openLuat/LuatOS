## 功能模块介绍

1、main.lua：主程序入口；

2、sht30_app.lua：每隔1秒读取一次温湿度数据；

3、AirSHT30_1000.lua：AirSHT30_1000驱动文件；

## 演示功能概述

AirSHT30_1000是合宙设计生产的一款I2C接口的SHT30温湿度传感器配件板；

本demo演示的核心功能为：

Air780EPM开发板+AirSHT30_1000配件板，每隔1秒读取1次温湿度数据；

1、搭建好演示硬件环境

3、Luatools烧录内核固件和demo脚本代码
