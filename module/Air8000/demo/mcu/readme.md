## 演示模块概述

1、main.lua：主程序入口；

2、mcu_test.lua：MCU功能测试模块；

## 演示功能概述

使用Air8000核心板测试MCU相关功能，包括：

- MCU死机时的处理模式设置
- 唯一ID获取与显示
- 系统tick计数功能测试
- 64位tick计数和差值计算
- 微秒、毫秒、秒级别的时间计数
- 16进制字符串转换输出

## 演示硬件环境

1、Air8000核心板一块

2、TYPE-C USB数据线一根

## 演示软件环境

1、Luatools下载调试工具

2、[Air8000 V2016版本固件](https://docs.openluat.com/air8000/luatos/firmware/)
