# pinyin (拼音输入)

1、LuatOS支持pinyin (拼音输入)，通过输入汉字的拼音进行选字。

2、使用pinyin (拼音输入)开发的demo项目可以参考以下个demo：

Air8101核心板+AirLCD_1020配件板：支持exeasyui各种组件、支持矢量字体、支持触摸控制

- 代码：[module/Air8101/demo/ui/easyui/single](https://gitee.com/openLuat/LuatOS/tree/master/module/Air8101/demo/ui/easyui/single)
- 文档：[pinyin (拼音输入)在Air8101+AirLCD_1020上的应用](https://docs.openluat.com/air8101/luatos/app/multimedia/ui/single/)

在这个demo中，pinyin (拼音输入)的应用，详见**single**文件夹内的win_input.lua脚本文件
